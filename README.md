# Bootstrap Responsive Landing Page

This is my second major project as a Frontend Developer at **Betechified Africa**. The objective was to build a fully responsive landing page using the **Bootstrap 5** framework.

## 🌟 Project Highlights
This project was a significant challenge as it was my first time working with this CSS framework. 

- **The Challenge:** Transitioning from "vanilla" CSS to Bootstrap's grid system and components.
- **The Result:** A fully responsive website that adapts to mobile, tablet, and desktop screens.
- **Feedback:** My mentor was impressed with the layout and implementation, marking a successful first attempt at modern UI frameworks.

## 📈 My Journey
- **Timeline:** I had been coding for **4 weeks** when I built this.
- **Focus:** Learning how to read framework documentation and use pre-built components like Navbars, Cards, and the Grid system.

## 🛠️ Technologies Used
- HTML5
- Bootstrap 5 (CSS Framework)
- Custom CSS (for minor branding tweaks)
